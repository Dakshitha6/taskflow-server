## Getting Started
1.Please install softwares below to run server locally
-Nodejs
-Mongodb

2.To clone the server from github-
mkdir taskflow-server
git clone https://github.com/Dakshitha6/taskflow-server.git
cd taskflow-server

3.Later run
npm i

4.To run server in development mode
npm run start:dev

5.To run server in production mode 
npm run build
cd dist
npm start

## Deployed on Railway
Deployment is automated.Any changes in main branch,server will be deployed
